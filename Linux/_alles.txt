mkdir (make directory) - создание папки.
touch - создание файла.
cp (copy) - копирование файла или папки.
mv (move) - перемещение файла или папки.
rm (remove) - удаление файла или папки. Для удаления целых каталогов используется флаг -r (рекурсивное удаление).
whoami - узнать, под каким именем мы работаем в системе.
tree - просмотр дерева каталогов. Команда показывает структуру каталогов и их содержимое.
history - вывод истории введенных команд.
pwd (print working directory) - просмотр текущей рабочей директории.
export_file - скачать файл из песочницы.
cd (change directory) - команда навигации по каталогам.
ls (list) - просмотр содержимого каталога. Флаг -l дает детальную информацию, -a показывает все файлы, включая скрытые.
clear - очистить экран терминала.
date - показать текущую дату и время.
rmdir - удаление пустых папок.
rm -r - удаление иерархии с вложениями.
cp -r - копирование папки целиком.
mv (rename) - переименование файла или папки.
history > /tmp/history.txt - вывод истории команд в файл.

date - вывод текущей даты и времени.
echo - вывод текста или переменных на экран или в файл. Используется для дописывания в файл с >>.
cat - вывод содержимого файлов на экран.
wc (word count) - подсчет количества строк, слов и символов в файле. Используется с ключом -l для подсчета строк.
head - вывод первых строк файла. По умолчанию выводит 10 строк, можно изменить количество строк с помощью ключа -n.
tail - вывод последних строк файла. По умолчанию выводит 10 строк, можно изменить количество строк с помощью ключа -n. Ключ -f отслеживает изменения файла в реальном времени.
grep - поиск и фильтрация строк в файлах. Ключ -w ищет точное совпадение слова, -v исключает строки из вывода.
df (disk full) - отображение информации о дисковом пространстве. Ключ -h выводит данные в удобочитаемом формате.
awk - обработка текстовых данных. Используется для выбора столбцов из текстового файла или вывода другой команды.
sed (stream editor) - обработка текстовых данных. Используется для подстановки текста, выборочной печати текстовых файлов, редактирования текстовых файлов на месте.

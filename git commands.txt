Клонирование репозитория:
Откройте терминал или командную строку и выполните команду для клонирования репозитория с GitHub на ваш ноутбук. Для этого выполните следующую команду, заменив USERNAME на ваше имя пользователя GitHub и REPOSITORY_NAME на название репозитория:

bash
Копировать код
git clone https://github.com/USERNAME/REPOSITORY_NAME.git
Перейдите в директорию проекта:
После клонирования перейдите в директорию проекта с помощью команды cd:

bash
Копировать код
cd REPOSITORY_NAME
Работа с проектом:
Теперь вы можете работать с вашим проектом. Вносите изменения, добавляйте файлы, коммитьте изменения и синхронизируйте их с GitHub.

Пример настройки для конкретного репозитория:

bash
Копировать код
git config user.name "Your Name"
git config user.email "your.email@example.com"
После настройки попробуйте снова выполнить ваш коммит:

bash
Копировать код
git commit -m "01-08-24 Laotop"

Добавление изменений:
После внесения изменений, добавьте их в индекс с помощью команды:

bash
Копировать код
git add .
Создание коммита:
Создайте коммит с описанием изменений:

bash
Копировать код
git commit -m "Описание изменений"
Отправка изменений на GitHub:
Для синхронизации изменений с GitHub выполните команду:

bash
Копировать код
git push origin main
Обратите внимание, что если ваша основная ветка называется не main, а, например, master, замените main на master.

Получение обновлений с GitHub:
Если на GitHub произошли изменения, которые нужно синхронизировать с вашим ноутбуком, выполните команду:

bash
Копировать код
git pull origin main



git add .
git commit -m "Автоматический коммит"
git push -u origin master

В HTML и CSS, display: flex; это свойство, которое применяется к контейнеру для того, чтобы использовать гибкую модель размещения элементов. Это очень удобный способ организовать элементы внутри контейнера, особенно когда нужно адаптивно располагать элементы на различных устройствах. Вот основные свойства, которые используются с display: flex;:

  • justify-content

    • Это свойство определяет, как браузер распределяет пространство между и вокруг элементов по главной оси контейнера (обычно это горизонтальная ось). Оно может принимать следующие значения:
    • Это свойство определяет, как браузер распределяет пространство между и вокруг элементов по главной оси контейнера (обычно это горизонтальная ось). Оно может принимать следующие значения:
    • flex-start: элементы выравниваются по началу контейнера.
    • flex-end: элементы выравниваются по концу контейнера.
    • center: элементы центрируются в контейнере.
    • space-between: элементы распределяются равномерно в контейнере, первый элемент находится у начала контейнера, последний — у конца.
    • space-around: элементы равномерно распределяются в контейнере с равными отступами вокруг них.
    • space-evenly: элементы распределяются так, что расстояние между любыми двумя соседними элементами, а также до начала и конца контейнера, одинаковое.

  • align-items

    • Это свойство определяет выравнивание элементов в поперечной оси контейнера (обычно это вертикальная ось). Оно может принимать следующие значения:
    • flex-start: элементы выравниваются по началу поперечной оси.
    • flex-end: элементы выравниваются по концу поперечной оси.
    • center: элементы центрируются по поперечной оси.
    • baseline: элементы выравниваются по базовой линии текста.
    • stretch: элементы растягиваются, чтобы занять всю высоту контейнера (если их собственная высота не задана).

  • flex-direction

  • Это свойство определяет направление основной оси контейнера. Оно влияет на то, как элементы будут следовать друг за другом внутри контейнера. Оно может принимать следующие значения:
  • row: элементы располагаются по горизонтали, как обычный текст.
  • row-reverse: элементы располагаются по горизонтали в обратном порядке.
  • column: элементы располагаются вертикально.
  • column-reverse: элементы располагаются вертикально в обратном порядке.

Эти свойства позволяют создавать разнообразные и гибкие макеты без использования дополнительных элементов-оберток или сложных CSS-хаков. Важно помнить, что flexbox работает лучше всего в одномерных макетах (либо по горизонтали, либо по вертикали). Если вам нужно создать более сложный двумерный макет, возможно, стоит рассмотреть использование CSS Grid.
